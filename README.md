# Tic-Tac-Toe Game in Java

This is a simple **Tic-Tac-Toe game** implemented in **Java**. It allows two players to take turns in a **3x3 grid** and play the game until there is a winner or the game ends in a draw.

## Features:
- **Player vs Player** gameplay (Two players alternate turns).
- **Game Board** displayed after each move.
- **Winner check** after every move to determine if any player has won.
- **Input validation** to ensure valid moves.

## Technologies Used:
- **Java** (Core Programming Concepts)
  - 2D Arrays to represent the game board.
  - Loops, conditionals, and methods for game flow.
  - Input handling with `Scanner` for user input.

## What I Have Learned:
- **Working with 2D Arrays**: I used a 2D array to represent the game board, which required me to understand how to store and access data in two dimensions.
- **Loops & Conditionals**: Implemented loops to handle the repeated player moves and conditional statements to check for valid moves and game winners.
- **Game Logic**: I learned how to create the logic for determining a winner based on the rows, columns, and diagonals of the board.
- **Input Handling**: Used the `Scanner` class to handle user input and validate that players input valid moves.
- **Modular Programming**: Broke down the game into smaller methods, improving code readability and reusability.
- **Basic Error Handling**: Included error handling for invalid moves to ensure a smooth game experience.
